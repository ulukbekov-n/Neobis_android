*This product is an application that performs the functions of a Dice Roller.
This app allows users to roll a virtual device with just a tap of a button
*To launch this app it is necessary to crate an virtual device emulator on android visual studio
or have a device on tha android operating system.
*![](../Naken/Без имени.png)
*This application can replace real cubes in different games,
thereby helping to save time and money.
